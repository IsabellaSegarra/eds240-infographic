# Microplastics in the San Francisco Bay 
This analysis utilizes data from San Francisco Estuary Institute regional study of microplastic pollution of a major urban estuary and adjacent ocean environment, sampling in stormwater, treated wastewater, surface water, sediment, and prey fish. 

This repository's purpose is for the EDS 240 final project. This project was completed as part of [EDS-240 Data Visualization and Communication](https://eds-240-data-viz.github.io/) class as part of the Masters of Environmental Data Science program.
